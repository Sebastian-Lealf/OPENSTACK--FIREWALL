# LABORATORIO DE UN FIREWALL CON OPENSTACK Y DEBIAN

Guia muy breve del laboratorio de firewall sobre OpenStack y Debian.

- Docente: Jorge Eliecer Gomez Gomez
- Integrantes: Diaz Cabarcas Camilo Andres; Garcia Molina Javier Andres; Gonzalez Echavarria Andres Camilo; Leal Florez Sebastian Jose
- Enfoque: montar VirtualBox + Debian, desplegar OpenStack/DevStack, crear redes y VMs, aplicar UFW (deny incoming/allow outgoing) con reglas a puertos 22/80/443/8080 y subred interna 192.168.100.0/24, generar trafico de prueba, capturar con tcpdump/Wireshark y verificar bloqueos/permisos.
- Entregables: `Laboratorio Firewall Openstack.pdf` y video explicativo https://youtu.be/rhATDoH_ibc
- Universidad de Cordoba – Facultad de Ingenierias – Ingenieria en Sistemas – Monteria, Cordoba – 2025
